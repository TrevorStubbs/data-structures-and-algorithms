## Reverse an Integer

- Author: Trevor Stubbs

---

### Problem Domain
Build a method that reverses all the digits in an integer.

---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| [123] | [321] |
| [-321] | [-123] |


---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
