## Integer is a Palindrome

- Author: Trevor Stubbs

---

### Problem Domain
Return a true if the given integer is a palindrome

---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| [121] | true |
| [-121] | false |


---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
